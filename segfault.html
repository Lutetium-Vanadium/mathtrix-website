Segfault

A game of logic, skills and unique thinking. Segfault is a COVID-19 themed hackathon event that tests the application of statistics and probability in Computer Science. Participants need to employ their creativity to design an application that makes sense of COVID related data to produce meaningful information.

The event starts 5 days before Mathtrix day, teams must consist of 2-5 players. Each team needs to create an application of their choice based on the given dataset, which the team will need to present and answer questions(based on their on project) on Mathtrix day. Team members must know one of the listed languages, as well how to use GitHub.

Rules

  • Team members require a GitHub account and should submit their usernames on the first day.
  • All code must be in a public GitHub repository and link for the same should be submitted on day 1. Code must be commited regularly, not in large chunks.
  • The following languages will be allowed:
    • Python
    • Javascript/Typescript
    • Swift
    • Objective-C
    • Java
    • Kotlin
    • Rust
    • Dart
    • C
    • C++
    If you want to use any other language, please contact the event heads.
  • Participants will need to create a presentation and demo their project on the final day. The presentation must convey the purpose of the application and the basic technical structure behind it. Questions may be asked based on the presentation and code base.
